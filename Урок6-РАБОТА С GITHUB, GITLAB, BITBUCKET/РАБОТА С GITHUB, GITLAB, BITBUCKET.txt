1. В чём отличие собственно git от github?
   Git - это локальное программное обеспечение, которое позволяет разработчикам сохранять моментальные снимки своих проектов с течением времени. 
   Как правило, он лучше всего подходит для индивидуального использования. GitHub-это веб-платформа, которая включает в себя функции контроля версий git, 
   чтобы их можно было использовать совместно. Кроме GitHub существуют и другие платформы такие как гитлаб, битбакет и др.
2. В чём отличия различных хранилищ?
   Фактически это одно и тоже, только разные компании владельцы.
3. Обязателен ли файл readme?
