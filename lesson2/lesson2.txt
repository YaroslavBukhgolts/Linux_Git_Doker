1. С какого ключевого слова должен начинаться любой докер файл?
   Любой dockerfile должен начинаться с клчевого слова FROM.
2. Каким образом происходит обмен данными с контейнером, если он представляет собой 
   закрытую структуру? 
   Обмен происходит с помощью команды VOLUME, которая создает точку монтирования для работы с постоянным хранилищем.
3. Каким образом можно избежать длительного процесса запуска npm ci при изменении 
   определённых файлов проекта из за подгрузки каждый раз всех слоёв докер-контейнера?
   Запускать сборку проекта как можно ближе к концу докер файла после копирования package фалов и запуска npm ci.
