1. SSH - (Secure Shell) - сетовой протокол для удаленого подключения. 
   В основном ssh используется для удаленного управления серверами через терминал.
2. Важной особенностью SSH является, его способность передачи любого сетевого протокола через 
   специализированный шифрованный канал связи (тунелирование). Благодаря этому у пользователя появляется не просто 
   возможность удаленно работать на личном ПК, но и передавать текстовые, графические, 
   аудио - и видеопотоки по защищенному шифрованному каналу.
3. Для ознакомления мануала по использованию ssh используется команда в терминале man ssh
4. Для генерирования ssh ключа используется команда ssh-keygen -t rsa
