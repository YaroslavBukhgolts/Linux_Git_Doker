1. Какая команда позволяет инициализировать проект в git?
   Инициализировать проект в git позволяет команда git init
2. Как произвести коммит файла и для чего это вообще нужно?  
   Чтобы произвести коммит необходимо использовать команду git commit. Команда нужна чтобы фиксировать измения сделаные в проекте.
3. Как добавить все файлы в отслеживание git?
   Чтобы добавить все файлы для отлеживания git необходимо использовать команду git add *
